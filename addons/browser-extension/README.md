# OTRUST Browser Extension

Source for the OTRUST browser extension.

The extension can hash page content locally and submit timestamping requests to the OTRUST API. It does not include Chrome Web Store assets, generated packages, or store screenshots.

## Local Loading

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Select "Load unpacked".
4. Choose this directory.

The extension targets Manifest V3.
