Run your own.
Don't trust ours.

Use otrust.eu on the web — or run your own instance in 60 seconds. Zero-knowledge: we only see hashes, never your files.

Step 1 — 60 seconds

git clone https://github.com/otrust-eu/opensource.git
cd opensource
./scripts/quickstart.sh

Step 2 — API keys & webhooks

Open /developers.html on your instance. Create an org with your admin key. Issue API keys. Register webhooks.

Developer portal

Step 3 — Every release, timestamped on Bitcoin

# .github/workflows/timestamp-release.yml
# Copy from examples/github-action/timestamp-release.yml

secrets.OTRUST_API      → https://otrust.yourcompany.com
secrets.OTRUST_API_KEY  → from developer portal
Self-host rule: never set HOSTED_MODE. Plan limits apply only on the hosted service at otrust.eu. Your instance — your rules, unlimited platform features.
