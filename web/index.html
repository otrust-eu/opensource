Platform

Four product surfaces, one verification layer.

OTRUST now covers the full trust journey: timestamp a record, collect signatures, create an identity proof, and use that proof in a global eID-ready partner-branded sign-in flow.

01 Timestamp

Hash locally, anchor externally, verify later.

 02 Signed

Collect intent and signatures with proof metadata.

 03 Proof

Create reusable identity proof packages.

 04 Sign in

Authenticate from a proof with partner branding for global identity flows.

Integrations and developer surfaces

Connect OTRUST where teams already work.

Web Tool

Timestamp files directly in your browser. Hash computed locally, never uploaded.

CLI Tool

Command-line interface for automation and scripting. Batch timestamp hundreds of files.

Verification

Verify any timestamp against blockchain. Independently auditable proofs.

Self-Hosting

Run the open source OTRUST server, web UI, SDKs, CLI, and circuits from GitHub.

Browser Extension

Timestamp any webpage with one click. Install from Chrome Web Store.

ChatGPT Integration

Timestamp conversations directly in ChatGPT. Just ask the GPT to timestamp your chat.

Email Timestamper

Send an email to timestamp@otrust.eu. Get a receipt and proof links back.

Telegram Bot

Timestamp messages and files directly in Telegram. Works in groups too.

WhatsApp Bot

Enterprise WhatsApp Business API integration. Contact us for setup.

Hosted Sign In

Use OTRUST identity proofs for passwordless auth with approved partner branding.

SDK Playground

Interactive playground to test the TypeScript/JavaScript SDK. Try all API functions live.

REST API Docs

Full API documentation with examples. OpenAPI spec available for code generation.
