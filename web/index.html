Built on principles

Trust is
earned by design.

OTRUST exists to make digital truth verifiable without permission, accounts or intermediaries.

Local-first

Hashing and sensitive checks run in your browser where possible.

Zero-knowledge

Timestamping never needs your file contents.

Immutable

Anchored in the Bitcoin blockchain via OpenTimestamps.

Open source

Transparent by default. Verifiable by everyone.
